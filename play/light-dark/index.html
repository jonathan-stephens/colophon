Overview

This system uses three complementary color scales per theme:

  • Brand colors: Primary interactive elements (buttons, links, focus states)
  • Neutral colors: Structural elements (backgrounds, text, borders)
  • Accent colors: Secondary actions, highlights, and informational elements

Try it: Switch themes and toggle dark mode to see how the three color scales work together harmoniously.

How It Works

Step 1: Define Three Color Scales

body[data-theme="ocean"] {
  /* Brand: Cobalt Blue */
  --brand-lightest: #eff6ff;
  --brand-mid: #3b82f6;
  --brand-darkest: #172554;
  
  /* Neutral: Muted Navy */
  --neutral-lightest: #f8fafc;
  --neutral-mid: #64748b;
  --neutral-darkest: #1e293b;
  
  /* Accent: Robin's Egg */
  --accent-lightest: #ecfeff;
  --accent-mid: #06b6d4;
  --accent-darkest: #164e63;
}

Step 2: Create Semantic Mappings

body {
  color-scheme: light dark;
  
  /* Neutral for structure */
  --bg-page: light-dark(
    var(--neutral-lightest), 
    var(--neutral-darkest)
  );
  
  /* Brand for interactions */
  --interactive-brand: var(--brand-mid);
  --interactive-brand-hover: light-dark(
    var(--brand-dark), 
    var(--brand-light)
  );
  
  /* Accent for highlights */
  --interactive-accent: var(--accent-mid);
}

Step 3: Use Semantic Variables

.btn-primary {
  background: var(--interactive-brand);
}

.btn-accent {
  background: var(--interactive-accent);
}

.card {
  background: var(--bg-surface);
  border: 1px solid var(--border-default);
}

Key Benefits

Clear Visual Hierarchy

Brand draws attention to primary actions, Accent highlights secondary information, and Neutral provides structure.

Semantic Clarity

Variables describe purpose (--interactive-brand) not appearance (--color-blue).

Maintainability

Change themes by swapping a single attribute. Add new themes without touching component CSS.

Accessibility

Respects OS-level dark mode preference. Consistent contrast ratios across themes.

Browser Support

The light-dark() function is supported in Chrome/Edge 123+, Firefox 120+, and Safari 17.5+.
