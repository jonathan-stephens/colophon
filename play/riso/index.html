Multi-Layer Color Separation Technique

This is THE technique for authentic risograph effects. Stack multiple colored divs with individual filters:

<div class="riso-layers">
  <div class="riso-layer layer-blue layer-1"></div>
  <div class="riso-layer layer-red layer-2"></div>
  <div class="riso-layer layer-yellow layer-3"></div>
</div>

Each layer should have:

  • position: absolute to stack layers
  • mix-blend-mode: multiply to blend like ink
  • Unique filter: url(#riso-grain-color) for grain
  • Small transform: translate() for registration shift
  • CSS gradients, shapes, or images as content

1. Authentic Riso Colors

  • Blue: #0078BF
  • Red: #FF5C5C
  • Yellow: #FFE800
  • Green: #00A95C
  • Purple: #765BA7
  • Fluorescent Pink: #FF48B0

2. Layer Registration Shifts

.layer-1 { transform: translate(1px, 0.5px); }
.layer-2 { transform: translate(-0.5px, 1px); }
.layer-3 { transform: translate(0.5px, -0.5px); }

3. Ink Overlap Effects

The magic happens where colors meet. mix-blend-mode: multiply is essential:

.riso-layer {
  mix-blend-mode: multiply;
}

Blue + Yellow = Green, Red + Yellow = Orange, Red + Blue = Purple

4. Grain Variation

Each color should have slightly different grain for realism:

  • Use different seed values in feTurbulence
  • Vary baseFrequency between 0.8-1.2
  • Lighter colors (yellow) can have less grain

5. Creating New Color Filters

To create a new color filter, duplicate an existing one and modify:

<filter id="risograph-NEWCOLOR">
  <!-- Change seed values (make them unique) -->
  seed="200" 
  
  <!-- Adjust the color matrix -->
  <!-- Example for orange: -->
  values="1.2 0.6 0 0 0.05    /* Red channel */
          0.6 0.8 0 0 0.02    /* Green channel */
          0 0.2 0.5 0 0       /* Blue channel */
          0 0 0 1 0"          /* Alpha channel */
</filter>

6. Performance Optimization

SVG filters can be GPU-intensive. For better performance:

.riso-layer {
  will-change: filter;
  transform: translateZ(0);
}

7. Using with Images

Apply filters to photos for a riso duotone effect:

<img src="photo.jpg" class="riso-blue" alt="Photo">

8. Responsive Design

Grain size stays consistent across screen sizes, but adjust height for mobile:

@media (max-width: 768px) {
  .riso-layers {
    height: 200px;
  }
}

Key Filter Parameters Explained

  • baseFrequency: Controls grain size (0.5 = coarse, 2.0 = fine)
  • numOctaves: Grain complexity/detail (2-6 range)
  • seed: Randomization value (change for different patterns)
  • stdDeviation: Blur amount for ink absorption (0.1-0.5)
  • tableValues: Controls grain visibility/opacity
  • slope: Contrast adjustment (1.0 = normal, 1.3 = high contrast)

ðŸŽ¯ Key Takeaway: True risograph realism comes from layering multiple filtered elements with mix-blend-mode: multiply, not from a single filter. Each layer should have its own grain pattern and slight positional offset. This mimics how real risograph printers apply one color at a time with mechanical imperfections.

What Each Filter Does:

  • #risograph-blue/red/yellow/green: Complete enhanced filters with dual grain + blur + contrast + color shift. Use these on single elements.
  • #riso-grain-blue/red/yellow: Lighter grain-only filters for multi-layer stacking. Use these when building color separations.
  • #risograph-enhanced: Ultimate filter with paper texture background. Most realistic for single elements.
